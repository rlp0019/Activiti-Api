Proyecto Fin de Grado - Activiti-Api

Grado en Ingeniería Informática, Universidad de Burgos

David Blanco Alonso y Roberto Luquero

La finalidad de este proyecto es la creación de una aplicación que permita un estudio del estado de un proyecto en GITHUB mediante distintas métricas, permitiendo tener una visión del ritmo de trabajo del proyecto, duración, numero de participantes y actividad.

También presenta diversas funcionalidades que permiten obtener conclusiones sobre el desarrollo del proyecto, analizando los datos de las métricas obtenidas.
